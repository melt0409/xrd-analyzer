XRD Parameter Reference

1. Axis Choices and Physical Meaning

1.1 2θ — the instrument's measured angle

Meaning: This is the raw diffraction angle directly measured by the diffractometer.

Limitation: 2θ depends strongly on the X-ray wavelength λ. If one dataset uses Cu Kα (λ ≈ 1.54 Å) and another uses Mo Kα (λ ≈ 0.71 Å), the same crystal will produce peaks at very different 2θ positions, so the two patterns cannot be directly overlaid without conversion.[1]

1.2 d-spacing — the crystal's fingerprint

Formula:

$$d = \frac{\lambda}{2\sin\theta}$$

Meaning: $d$ is the physical spacing between lattice planes in the crystal, usually expressed in angstroms (Å).

Use: This is an intrinsic property of the material. No matter what wavelength is used, the same crystal gives the same $d$-spacing values. That is why reference databases such as JCPDS or ICDD PDF cards are commonly matched using $d$-spacing rather than 2θ.[2]

1.3 q — the reciprocal-space language

Formula:

$$q = \frac{4\pi\sin\theta}{\lambda}$$

Meaning: $q$ is the magnitude of the scattering vector in reciprocal space, describing the momentum transfer during diffraction.

Use: In SAXS, WAXS, neutron diffraction, and synchrotron work, $q$ is often used as the standard x-axis because it allows data collected with different wavelengths to be compared on the same scale.[3]

1.4 sinθ/λ — the crystallographic working variable

Meaning: This is equivalent to $1/(2d)$, or $q/(4\pi)$.

Use: In crystallography, atomic scattering factors are often discussed as functions of $\sin\theta/\lambda$, making this a convenient standard variable for refinement and structure analysis.

This section summarizes the main formulas and crystallographic rules implemented in xrd_app.py.

2. Bragg's Law

Formula:

$$n\lambda = 2d\sin\theta \quad (n=1)$$

Where:

  • λ = X-ray wavelength (Å)
  • d = interplanar spacing (Å)
  • θ = Bragg angle, half of the measured 2θ

2.1 Uses in the code

d → θ: predicts where a peak should appear:

$$\theta = \arcsin\left(\frac{\lambda}{2d}\right)$$

Then the plotted peak position is 2θ. Implemented in calculate_bragg_angle(wavelength, d_spacing). If $\lambda/(2d) > 1$, the reflection is physically unreachable at that wavelength and the function returns None.

θ → d is not used directly, but the inverse relation is used when converting a measured pattern from one wavelength to another.

3. Interplanar Spacing: General Triclinic Formula

Because triclinic is the lowest-symmetry crystal system, its $d$-spacing formula is the most general one and can also describe higher-symmetry systems by applying symmetry constraints such as $\alpha = \beta = \gamma = 90^\circ$. For this reason, the code implements only the triclinic case in CrystalSystemCalculator.triclinic.

3.1 Unit-cell volume

$$V = abc\sqrt{1 - \cos^2\alpha - \cos^2\beta - \cos^2\gamma + 2\cos\alpha\cos\beta\cos\gamma}$$

3.2 Reciprocal spacing

$$\frac{1}{d^2} = \frac{h^2b^2c^2\sin^2\alpha + k^2a^2c^2\sin^2\beta + l^2a^2b^2\sin^2\gamma + 2hkabc^2(\cos\alpha\cos\beta - \cos\gamma) + 2kla^2bc(\cos\beta\cos\gamma - \cos\alpha) + 2hlab^2c(\cos\alpha\cos\gamma - \cos\beta)}{V^2}$$

Where:

  • a, b, c = unit-cell edge lengths
  • α, β, γ = unit-cell angles
  • h, k, l = Miller indices

If the square-root term in the volume expression becomes negative due to invalid geometry, it is clamped to zero. If $1/d^2 \le 0$, the reflection is treated as nonexistent, with $d = \infty$. Implemented in CrystalSystemCalculator.triclinic(h, k, l, a, b, c, alpha_deg, beta_deg, gamma_deg).

4. Systematic Absences

A reflection allowed by Bragg's law may still be absent because of crystal symmetry. These are called systematic absences. The program checks three independent sources of absence; a reflection is marked absent if any rule forbids it.[4]

4.1 Lattice Centering

Lattice typeSymbolCondition for reflection to be allowed
PrimitivePalways allowed
Body-centeredI$h+k+l = 2n$
Face-centeredF$h, k, l$ all even or all odd
A-centeredA$k+l = 2n$
B-centeredB$h+l = 2n$
C-centeredC$h+k = 2n$
H-centered (hexagonal obverse)H$(h-k) \bmod 3 = 0$
R-centered, obverse settingR (obv)$(-h+k+l) \bmod 3 = 0$
R-centered, reverse settingR (rev)$(h-k+l) \bmod 3 = 0$

Implemented in lattice_forbidden(h, k, l, lattice_type).

4.2 Screw Axes

A screw axis affects only axial reflections along its own axis: $h00$ for the $a$-axis, $0k0$ for the $b$-axis, and $00l$ for the $c$-axis. For such reflections, the index along the screw axis must satisfy:

Screw axisCondition for the axial index to be allowed
$2_1, 4_2, 6_3$index is even
$3_1, 3_2, 6_2, 6_4$index mod 3 = 0
$4_1, 4_3$index mod 4 = 0
$6_1, 6_5$index mod 6 = 0

Implemented in screw_axis_forbidden(h, k, l, axis, screw_rule).

4.3 Glide Planes

A glide plane perpendicular to an axis affects only zonal reflections with that axis index equal to zero: $0kl$ perpendicular to $a$, $h0l$ perpendicular to $b$, and $hk0$ perpendicular to $c$. For the remaining two indices $u$ and $v$:

Glide typeCondition for u, v to be allowed
a-glide$u$ is even
b-glideindex along $b$ is even
c-glideindex along $c$ is even
n-glide$u+v$ is even
d-glide$u+v$ is divisible by 4

Implemented in glide_plane_forbidden(h, k, l, plane_perp_axis, glide_rule).

4.4 Combining the Rules

A reflection is classified as:

  • absent if lattice_forbidden OR screw_axis_forbidden OR glide_plane_forbidden
  • allowed otherwise

Implemented in systematic_absence_reasons(...) and classify_reflection(...).

5. Wavelength Conversion of a Measured Pattern

When a pattern measured at wavelength $\lambda_1$ needs to be displayed as if it were measured at wavelength $\lambda_2$ — for example, converting Cu Kα₁ data to Co Kα — the $d$-spacing of each peak remains physically constant. Bragg's law is therefore applied twice:

$$\lambda_1 = 2d\sin\theta_1, \quad \lambda_2 = 2d\sin\theta_2$$

so

$$\sin\theta_2 = \left(\frac{\lambda_2}{\lambda_1}\right)\sin\theta_1$$

and therefore

$$\theta_2 = \arcsin\left[\left(\frac{\lambda_2}{\lambda_1}\right)\sin\theta_1\right]$$

which gives

$$2\theta_2 = 2\arcsin\left[\left(\frac{\lambda_2}{\lambda_1}\right)\sin\left(\frac{2\theta_1}{2}\right)\right]$$

If $|(\lambda_2/\lambda_1)\sin\theta_1| > 1$, the peak has no valid angle at the new wavelength and is marked invalid (NaN). Implemented in _convert_two_theta_by_wavelength(two_theta_deg, lam_from, lam_to).

6. Zero-Shift Correction

A simple additive correction is applied to compensate for instrument misalignment:

$$2\theta_{\text{corrected}} = 2\theta_{\text{converted}} + \Delta 2\theta$$

where $\Delta 2\theta$ is the zero-shift offset in degrees. Applied throughout compute_and_plot(), export_to_txt(), and save_project().

7. Overlay Window

These are display-only transformations used for comparing multiple curves; they are not physical formulas.

7.1 Normalization

$$y_{\text{norm}} = \frac{y - y_{\min}}{y_{\max} - y_{\min}}$$

7.2 Vertical scaling and shifting

$$y_{\text{display}} = y \cdot \text{scale} + y_{\text{shift}}$$

7.3 Interactive drag-to-scale

Dragging a curve up or down rescales it so its peak reaches the new height:

$$\text{scale} = \frac{\text{peak\_value} + y_{\text{shift}}}{\text{peak\_value}}$$

Implemented in _compute_display_y() and _on_b_window_motion().
